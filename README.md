

O projeto "Sáude já" é um aplicativo desenvolvido em React Native, com o objetivo de facilitar o acesso a informações de saúde e funcionalidades internas relacionadas ao acompanhamento do usuário.
O projeto utiliza uma estrutura modular com componentes reutilizáveis, navegação organizada e gerenciamento simples de estado.

Tecnologias utilizadas

- JavaScript 
- React Native
- Expo
- React Hooks (useState, useEffect)
- Assets internos para interface (imagens e ícones)
- Geolocation
 Estrutura do projeto

O app está organizado da seguinte forma:

- src→ telas, componentes e lógica principal
- assets → imagens e ícones do app
- App.js → ponto inicial do app
- Configurações: app.json, package.json, metro.config.js

Como executar

bash
 Clone o repositório
git clone https://github.com/ClaraqueirozM/saude-ja-app.git
cd saude-ja-app

 Instale as dependências
npm install

 Inicie o app
npm start   # ou expo start
